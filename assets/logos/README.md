# Palma OS Logo Assets

This folder contains the official branding assets for Palma OS.

## Files
- `palma-logo.svg` - Main logo (vector)
- `palma-logo-128.png` - 128x128 PNG
- `palma-logo-256.png` - 256x256 PNG
- `palma-logo-512.png` - 512x512 PNG
- `palma-icon.svg` - App icon style logo

## Color Specifications
- Primary Green: #009B77
- Gold Accent: #FFD700
- White: #FFFFFF
- Dark text: #333333

## Usage Guidelines
1. Always use the logo on light backgrounds
2. Maintain adequate spacing around the logo
3. Do not distort or modify the logo proportions
